## 2. 毎朝9時に全自動で動くもの

Windowsタスクスケジューラ **`CARJAM_DailyUpdate`** が `scripts\daily-run.ps1` を実行。
中身は以下を順番に呼ぶだけ：

| # | スクリプト | 内容 | 頻度 |
|---|---|---|---|
| 1 | `check-events.ps1` | **17サイト**をスクレイピングしてイベント収集 | 毎日 |
| 2 | `gen-blog.ps1` | **Claude API**で記事2本生成 → はてなブログ投稿 | 毎日 |
| 3 | `gen-pages.ps1` | イベント/記事の個別HTML + sitemap.xml 生成 | 毎日 |
| 4 | `gen-x-events.ps1` | Xにイベントまとめを投稿 | **偶数日のみ** |
| 5 | `scan-x-events.ps1` | Xでカーイベント告知を探しレポートを出す（**自動掲載はしない**） | **水曜のみ** |
| 6 | （daily-run内） | git add/commit/push → Netlifyデプロイ | 毎日 |

ログ: `scripts\run.log`（**UTF-16なので `Get-Content` で読む。bashのgrepは文字化けする**）

## 8. 既知のハマりどころ

| 事象 | 対処 |
|---|---|
| PowerShell 5.1 で `.Count` が `$null` | `@()` で配列化してから `.Count`（1件ヒット時のバグ）。7/3の記事重複の原因 |
| `"本文$var文字"` で変数が消える | PSは日本語も変数名に使えるので `$var文字` が1つの変数名になる。`"$($var)文字"` と書く |
| スクリプトを書いたら日本語が全部壊れた | **UTF-8 BOM無し**で保存された。PS5.1はcp932と誤読する。BOM付きで保存し直す |
| 地名判定にページ全体を渡すと誤判定 | ナビの「都道府県から探す」を拾う。判定は**イベント名だけ**に渡す。7/20に52件誤りの原因 |
| 画面に出す数字が実態とズレる | **数字は必ずデータから計算する。HTMLに手打ちしない。** 7/20まで「27都道府県」「126+件」が固定値だった |
| 件数表示と実際の表示件数が食い違う | 絞り込みの件数を出すときは、**他のフィルタ（月・カテゴリ・検索）も効かせて数える**。でないと「徳島県（3）」を選んで0件になる |
| 新しい収集元を足す前に | **robots.txt だけでなく利用規約の本文を読む。** JAF・走行会.com は規約で無断使用を禁止していた（→§7の不採用リスト） |
| 外部カレンダーに内部向け予定が混ざる | JMRC四国のICSには「総会」「運営委員会」「Aライセンス講習会」が入っていた。一般来場者が行けないものは除外する |
| `run.log` が文字化け | UTF-16。`Get-Content` で読む |
| `config.ps1` が cp932 になる | メモ帳保存が原因。UTF-8 BOMに戻す |
| `ConvertFrom-Json` が配列を1個に潰す | `| ForEach-Object { $_ }` で展開 |
| `-ExecutionPolicy Bypass` がauto modeで拒否される | ユーザーに承認してもらう。ポリシーは全スコープUndefined |
| ヒーローの「掲載イベント数」が実数と違う | **仕様**（DB全件＝過去含む）。直さない |

## 11. SNSからのイベント収集について（2026-07-20 調査）

「インスタやXからも情報を集めたい」という要望を受けて調べた結果。

### ✅ 実装済み: `scan-x-events.ps1`（毎週水曜・自動掲載はしない）

**既存の `X_API_KEY` から Bearer Token を自動生成**するので、config.ps1 への追加設定は不要。

出力: `note-drafts\x-scan-YYYY-MM-DD.txt`（.gitignore済み）
→ **中身を見て、載せる価値があれば `submit.html` から登録する**

**なぜ自動掲載にしなかったか（実測値）**

87件を検索して、実際に使えた新規イベントは **1件だけ**だった:

| 段階 | 件数 |
|---|---|
| 検索でヒット | 87 |
| 日付＋都道府県が本文にある | 7 |
| 二輪を除外 | −4（PMCイベント、ハードエンデューロ、カワサキプラザ、トライアンフ） |
| ラジコンを除外 | −1（`#ラジドリ #RCドリフト #ミニッツ`） |
| 既にCARJAMにあるものを除外 | −1 |
| **使えた** | **1**（夏休みの初オフ体験走行会・いなべモータースポーツランド） |

Xの投稿は**日時・会場が画像のフライヤーにしか書かれていないことが非常に多い**。
画像内の文字はAPIでは取れない。そのまま取り込むと「会場・日付が未定」のレコードが
サイトに流入する（＝2026-07-20 に苦労して減らしたものが元に戻る）。

**除外処理は必須**。「走行会」は二輪でも使う語で、初回は候補7件中4件がバイクだった。
ラジコンは「MRD走行会」のように名前では判別できず、ハッシュタグでしか分からない。

**費用**: 1回あたり約65円（87件・$0.005/件）。毎日動かすと月2,000円規模になるため水曜のみ。

### X API について（2026-07 調査時点）
